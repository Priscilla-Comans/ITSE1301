Lab Four Part A

In this lab, we worked with Masks and Corrections. For lab four-part A, we had to take two images to create one. We need to create a mask to cover part of the second image. To make both images seem like one, the edges for the mask needed to be smooth out by refining the edges, blur, or outside glow Photoshop tools. The reason I chose these two images was that I own both photos.


The steps that I used to fix this image were the following:

  1. I set the train track image as the background and set the second image as layer one
  2. Resized images to 800 pixels in width and height.
  3. Added a white mask to layer 1.
  4. Used the brush tool to remove elements from layer one in order to expose the train tracks.
  5. Defined the blur edges by setting the radius to 4 pixels, enabling smart radius, added a 2% contrast, and 2% shift.
