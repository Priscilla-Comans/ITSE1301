Lab Five

Lab five requires that I created a poster by using vector shapes and text blocks. For this lab, we had to resize the image to be poster size. We had to wrap the Text to create visual effects and use paths to move or bend the Text. Filters were another requirement for this lab. This lab required a lot of creativity using serval Photoshop text tools.


The steps that I used to fix this image were the following:

  1. Resized to width: 11 inches and height: 17 inches.
  2. Brightness:-65 - Contrast:45.
  3. Vibrance :+88 - Saturation:+1.
  4. Windows - Shapes - Costume Shape Tool - legacy Shape and More - Fill:none - Stroke - Dark Folder - Brown - 4.43px - Solid Stroke - Shape - Farm Animals - Chicken - Horizontal type Tool - Veranda - Bold - Italic - 48pt - Centered Text - Eyedropper Tool - Picked color from wood-chips.
  5. Duplicated Chicken Layer - Resized Stroke to 5.71px - Color - Black - Changed Font Color - Black.
  6. Staggered layers to have the black font as the bottom layer.
  7. Added Rooster Shape - Fill:Eyedropper Tool - Picked color from blueish eggs - Stroke - White - 4.43px - Solid Stroke.
  8. Added Ellipse Shape - 0% Opacity - Horizontal Type Mask Tool - Myriad Variable Concept - Black - Semi Extended - 60 pt - Sharp - Hue:-2 - Saturation:+54 - Lightness:62.
  9. New Shape - Flag - Fill:None - Stroke: Black - 4.43px - Horizontal Type Mask Tool - Myriad Variable Concept - Black - Semi-bold Italic - 36pt - Font Color - White.
  10. Duplicated layer - Changed Font Color - Black - Changed Stroke Color- White
  11. Merged layers.
  12. Mixer Brush Tool - Kyle's Concept Brush - Foliage Mix 2 - 120px - Color - #f6af63 - Leafs Around Flag Shape.
