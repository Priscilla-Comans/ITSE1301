Lab Four Part B

For this lab, I chose this image because it was the one that looked most challenging for me. For part b of lab four, we need to correct light and contrast. To fix the color of this image, I was required to use the following Photoshop tools;

  • Levels
  • Curves
  • Histogram
  • Shdows/Highlights


The steps that I used to fix this image were the following:

  1. Changed width to 800 and height to 750.
  2. Used the curves tool to adjust the brightness of the colors. Green(input 27, output 52) Red(input 24, output 56) Blue(input 26, output 41) RGB(input 36, output 43).
  3. Shadows (-3,-4,+10).
  4. Brightness 8 and contrast -2.
